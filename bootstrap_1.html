Key Takeaway:

The infix (sm, md, lg) determines at what minimum screen size the columns will stop stacking vertically and begin appearing side by side.

Remember: Bootstrap uses a mobile-first approach, so columns stack by default and the grid classes determine when they should appear horizontally.

Practical Recommendation:

col-md-* is the most commonly used class as it provides a good balance between mobile and desktop experiences. Use col-sm-* when you want more horizontal layouts on phones, and col-lg-* when you want to prioritize vertical scrolling on smaller devices.
